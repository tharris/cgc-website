NAME

WormBase::API::Object::Paper

SYNPOSIS

Model for the Ace ?Paper class.

URL

http://wormbase.org/resources/paper

METHODS/URIs

Overview

name

This method will return a data structure containing the name and ID of the paper object.

PERL API
 $data = $model->name();
REST API

Request Method

GET

Requires Authentication

No

Parameters

a paper ID (eg WBPaper00000031)

Returns

  • 200 OK and JSON, HTML, or XML

  • 404 Not Found

Request example

curl -H content-type:application/json http://api.wormbase.org/rest/field/paper/WBPaper00000031
