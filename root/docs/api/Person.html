NAME

WormBase::API::Object::Person

SYNPOSIS

Model for the Ace ?Person class.

URL

http://wormbase.org/resources/person

METHODS/URIs

Overview

name

This method will return a data structure of the name and ID of the requested object.

PERL API
 $data = $model->name();
REST API

Request Method

GET

Requires Authentication

No

Parameters

A class and object ID.

Returns

  • 200 OK and JSON, HTML, or XML

  • 404 Not Found

Request example

curl -H content-type:application/json http://api.wormbase.org/rest/field/[CLASS]/[OBJECT]/name

Response example

street_address

This method returns a data structure containing the street address of the person, if known.

PERL API

 $data = $model->street_address();

REST API

Request Method

GET

Requires Authentication

No

Parameters

WBPerson ID

Returns

  • 200 OK and JSON, HTML, or XML

  • 404 Not Found

Request example

curl -H content-type:application/json http://api.wormbase.org/rest/field/person/WBPerson242/street_address

Response example
 { "name"           : "WBPerson242",
   "class"          : "person",
   "uri"            : "http://todd.wormbase.org/page/person/WBPerson242";}                               
   "street_address" : { "data" : [ "Department of Neuroscience",
                                   "Brown University",
                                   "185 Meeting Street, SFH458",
                                   "Mailbox GL-N",
                                   "Providence, RI 02912"
                                 ],
                         "description" : "street address of the person"
                      },
 }
