NAME

WormBase::API::Object::Feature

SYNPOSIS

Model for the Ace ?Feature class.

URL

http://wormbase.org/species/feature

METHODS/URIs

Overview

name

This method will return a data structure of the name and ID of the requested object.

PERL API
 $data = $model->name();
REST API

Request Method

GET

Requires Authentication

No

Parameters

A class and object ID.

Returns

  • 200 OK and JSON, HTML, or XML

  • 404 Not Found

Request example

curl -H content-type:application/json http://api.wormbase.org/rest/field/[CLASS]/[OBJECT]/name
