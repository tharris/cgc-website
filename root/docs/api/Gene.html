NAME

WormBase::API::Object::Gene

SYNPOSIS

Model for the Ace ?Gene class.

URL

http://wormbase.org/species/gene

METHODS/URIs

Overview

name

This method will return a data structure of the name and ID of the requested object.

PERL API

REST API

Request Method

GET

Requires Authentication

No

Parameters

A class and object ID.

Returns

  • 200 OK and JSON, HTML, or XML

  • 404 Not Found
